# ContentBay

A secure cloud-based content sharing platform built using React, Node.js, MongoDB, and AWS.

## Features

- User Authentication
- JWT Authorization
- Content Upload and Management
- Admin Dashboard
- Secure REST APIs
- AWS Deployment

## Technology Stack

### Frontend
- React
- TypeScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

### Cloud & DevOps
- AWS EC2
- Nginx
- PM2
- GitHub Actions

## Installation

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
